Mensagens: Resumo

O que é

Mensagens é uma ferramenta de comunicação que permite que os participantes do site se comuniquem usando e-mail do curso interno. Além disso, ela suporta mensagens entre os grupos.

Principais conceitos

Visualizar mensagens: As mensagens não lidas estão em negrito; mensagens visualizadas são marcadas como lidas, mas você pode marcar qualquer mensagem lida como não lida. Por padrão, as mensagens são listadas em ordem cronológica, mas você pode selecionar Por Conversa para exibir mensagens em tópicos recuados.

Compor mensagens: Uma interface WYSIWYG (What You See Is What You Get) permite texto rico, texto simples e de edição de HTML. Você também pode adicionar anexos.

Nota: Colar texto do Microsoft Word diretamente no editor WYSIWYG pode resultar na saída ilegível, mesmo que o texto aparecer correto no editor. Em vez disso, para inserir texto formatado a partir de um documento do Word, clique no ícone Colar do Word, coloque o cursor no "Colar do Word" subsequente caixa de texto e pressione Ctrl-v no Windows, ou Cmd-v no Macintosh, para colar o texto copiado. Use Ctrl-v (ou Cmd-v) para colar é o melhor método para preservar a formatação do documento do Word original. Clique em OK para adicionar o texto para o item que está compondo.

Mensagem privada: Você pode definir níveis de prioridade para mensagens para distinguir mensagens importantes de outros.

Considerações

  • Usando Mensagens, você pode enviar mensagens a outro indivíduo, um número restrito de indivíduos, ou um número restrito de grupos.

  • As mensagens que você enviar usando a opção Para Endereço de E-mail dos Destinatários são enviadas diretamente para endereços de e-mail dos participantes e não são armazenados na área de Mensagens.
