Viewing Discussion Forums

To see the list of Discussion Forums for your site, click Message Center in your site's menubar. On the Message Center screen, the Private Message Area is always on top, and the Discussion Forums area is on the bottom.

Note: The Discussion Forums area does not contain default forums. No forums will appear here until the site creator (or someone with an appropriate role) creates them.

Forums are listed with their titles and short descriptions (if created) displayed. Beneath the forum title, topics are listed by title, and include a short description (if created). A count of messages (total and unread) is shown for each topic. If a topic has an attachment, its filename and size will be displayed. Click the filename to open the attachment, or save it to your computer.

Clicking the name of any forum will open the Forum Detail view, which is an isolated view of a particular forum and its topics. Clicking the title of any topic will display all messages for that particular topic. If a full description of the topic exists (i.e., if the topic's author added the description in the Topic Settings screen), click Read Full Description to view it.

Depending on how the forum settings are configured, and your role within the site, the following options may be available:

  • New Forum opens the Discussion Forum Settings screen for creating a new forum. For more information, see Creating a new forum and topic in Discussion Forums.

  • New Topic opens the Topic Settings screen for creating a new topic. For more information, see Creating a new forum and topic in Discussion Forums.

  • Forum Settings opens the Discussion Forum Settings screen for revising an existing forum. For more information, see Revising or deleting a forum or topic.

  • Topic Settings opens the Topic Settings screen for revising an existing topic. For more information, see Revising or deleting a forum or topic.

  • Template Settings opens the Default Settings Template screen. For more information, see Revising or deleting a forum or topic.
